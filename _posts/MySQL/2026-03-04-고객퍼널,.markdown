퍼널분석: 사용자의 행동 경로를 단계별로 분석해, 각 단계에서 사용자가 어떻게 이동하고 이탈하는지 파악하는 분석 프레임워크  
온라인 쇼핑몰: 상품검색 > 장바구니 추가 > 구매  
마케팅: 브랜드 awarness > inserest > consideration > intent > purchase  
  
퍼널 분석의 중요성  
1. 고객 여정 이해: 제품/서비스에 접근하고 사용하는 전반적인 과정 이해  
2. 전환율 증진: 고객 여정 중 이탈하는 포인트와 원인 식별  
3. 마케팅 전략 최적화: 효과적인 마케팅 캠페인 식별  
  
```
-- SELECT * FROM `pj-1-489007.part1.funnel` LIMIT 1000

-- 퍼널 분석을 수행하기 위해 view, cart, purchased 별로 사용자 행동을 분석하고 이탈 지점 파악  
with funnel as (
  select 
  user_id,
  user_session,
  event_type, 
  event_time, 
  -- row number를 통해 이벤트 발생 순서를 정함
  ROW_NUMBER() OVER (partition by useR_id, user_session, event_type order by event_time) as event_rank
  from `pj-1-489007.part1.funnel`
  where
  event_type in('view', 'cart', 'purchase')
),
funnel_steps as (
  select 
  user_id,
  user_session,
  max(case when event_type = 'view' then event_rank end) as view_step,
  max(case when event_type = 'cart' then event_rank end) as cart_step,
  max(case when event_type = 'purchase' then event_rank end) as purchase_step
  from funnel
  group by 
  user_id, 
  user_session
),
-- 각 사용자가 최소 한번은 view, cart, purchase를 한번이라도 거쳤는지를 확인하고 각 단계별로 사용자 수 집계 
funnel_counts as(
  select 
  count(distinct user_id) as total_users,
  count(distinct case when view_step is not null then user_id end) as users_viewed,
  count(distinct case when cart_step is not null then user_id end) as users_carted,
  count(distinct case when purchase_step is not null then user_id end) as users_purchased
  from funnel_steps
)
-- 각 단계별 전환률 계산
select 
  total_users,
  users_viewed,
  users_carted,
  users_purchased,
  round(users_carted / users_viewed * 100, 2) as cart_conversion_rate,
  round(users_purchased / users_carted * 100, 2) as purchase_conversion_rate 
  from funnel_counts;
```
