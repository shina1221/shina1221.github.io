# 면접을 위한 CS 전공지식 노트  
> TCP/IP 4계층 모델

인터넷 프로토콜 스위트: 인터넷에서 컴퓨터들이 서로 정보를 주고받는데 쓰이는 프로토콜의 집합.  
이를 TCP/IP 4계층 모델로 설명하거나 OSI 7계층 모델로 설명하기도 한다.  

   TCP/IP 계층            OSI 7계층  
                     |  애플리케이션 계층     
애플리케이션 계층    |  프레젠테이션 계층   
                     |  세션 계층  
전송 계층            |  전송 계층  
인터넷 계층          |  네트워크 계층  
링크 계층            |  데이터 링크 계층  
                     |  물리 계층  
<br>  

애플리케이션 계층: FTP/HTTP/SSH/SMTP/DNS 등 응용 프로그램이 사용되는 프로토콜 계층.
