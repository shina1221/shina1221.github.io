> 깃에 올려놓은 이미지를 깃 블로그에 넣기위해선 다음의 마크다운 문법을 따라야함.
```
![GitHub Logo](/images/logo.png)
Format: ![Alt Text](url)
```
정확인 이미지 경로를 얻기 위해선
issues탭에서 submit new issue를 열어 git에 올려둔 이미지를 끌어놓으면 마크다운 문법이 보임. 

2차 문제해결  
git에서 오른쪽 마우스 버튼 클릭 후 이미지 주소 복사해서 다음의 서식을 따르면 됨
