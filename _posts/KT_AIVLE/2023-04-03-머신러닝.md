       time,    
       SUM(total_bill)

4) 주석을 쓸 것  
1. 서브쿼리 단위로 의미를 적음     
2. 컬럼 이름만 보고 해석에 오해가 있을 수 있거나, 추가 설명이 필요한 경우에 적어줄 것  
3. 조건이 왜 필요한지, 어떻게 동작하는지 같은 디테일을 적어주면 추후에 도움됨    
#MySQL에서의 주석은 --로 처리  

```MySQL
SELECT AVG(sales) avg_sales  
FROM (  
  -- 요일별 매출액 합계 계산  
  SELECT day -- 일별 X 요일별 O  
       , SUM(total_bill) sales  
  FROM tips  
  WHERE sex = 'Female' -- 결제자의 성별이 여성인 경우  
  GROUP BY day  
) daily_sales    
```
